# Arduino Code Examples

This folder contains example Arduino code files that can be loaded into the simulator's code editor.

## Available Examples

### 1. **blink.ino** - Basic Blink
The simplest Arduino program. Blinks the built-in LED (pin 13) on and off every second.
- **Difficulty**: Beginner
- **Components**: Arduino Uno (built-in LED)
- **Perfect for**: Testing the compiler and simulator

### 2. **blink_with_serial.ino** - Blink with Serial Output
Blinks LED and prints status messages to Serial Monitor.
- **Difficulty**: Beginner
- **Components**: Arduino Uno (built-in LED)
- **Perfect for**: Learning Serial communication

### 3. **potentiometer_read.ino** - Potentiometer Reading
Reads analog value from a potentiometer on pin A0 and displays it.
- **Difficulty**: Beginner
- **Components**: Arduino Uno, Potentiometer
- **Perfect for**: Learning analog input

### 4. **button_control.ino** - Button Control
Uses a button on pin 2 to control LED on pin 13.
- **Difficulty**: Beginner
- **Components**: Arduino Uno, Push Button, LED
- **Perfect for**: Learning digital input and pull-up resistors

### 5. **analog_threshold.ino** - Analog Threshold
Reads analog sensor and turns LED on when value exceeds threshold.
- **Difficulty**: Intermediate
- **Components**: Arduino Uno, Analog Sensor, LED
- **Perfect for**: Learning conditional logic with analog sensors

### 6. **motor_control.ino** - Motor Control
Controls a DC motor using PWM (Pulse Width Modulation).
- **Difficulty**: Intermediate
- **Components**: Arduino Uno, Motor Driver, DC Motor
- **Perfect for**: Learning PWM and motor control

### 7. **robot_control.ino** - Full Robot Control
Complete robot control system with motors, sensors, and LED.
- **Difficulty**: Advanced
- **Components**: Arduino Uno, Motors, Sensors, LED
- **Perfect for**: Learning integrated systems

## How to Use

1. Open the code editor by clicking "📝 Edit Code"
2. Select an example from the dropdown menu
3. The code will load into Monaco Editor
4. Click "▶️ Run" to compile and execute
5. Watch the simulation run!

## Testing the Compiler

Start with **blink.ino** - it's the simplest example and perfect for testing:
- Minimal code (easy to compile)
- No external components needed
- Clear visual feedback (LED blinking)
- Should compile without errors

## Notes

- All examples are compatible with Arduino Uno
- Examples use standard Arduino libraries only
- Code is well-commented for learning
- Each example demonstrates different concepts
